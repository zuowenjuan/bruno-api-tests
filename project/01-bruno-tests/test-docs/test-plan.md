\# 学生管理系统接口测试计划



&#x20;测试目标

验证所有接口功能正确，异常情况处理正确



&#x20;测试范围<br/>- 学生管理（4个接口）

\- 课程管理（4个接口）

\- 成绩管理（5个接口）

测试策略

2\. 异常流程：测试参数错误、权限错误

3\. 边界值：测试空值、超长值、特殊字符
