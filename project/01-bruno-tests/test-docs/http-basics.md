\# HTTP协议基础



1\. HTTP请求/响应流程

客户端（浏览器/Bruno）→ 发送请求 → 服务器<br/>客户端 ← 返回响应 ← 服务器



2\. 常用状态码

| 状态码 | 含义 | 示例 |

| 200 | 成功 | GET /api/users 返回用户列表 |

| 201 | 创建成功 | POST /api/users 创建用户 |

| 400 | 参数错误 | 缺少必填字段 |

| 401 | 未认证 | 没有token |

| 403 | 无权限 | token正确但权限不够 |

| 404 | 不存在 | GET /api/users/999 |

| 500 | 服务器错误 | 代码异常 |
