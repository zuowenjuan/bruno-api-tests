\# SQL进阶实战项目



\## 项目概述

基于学生成绩管理系统，实现20条SQL查询，覆盖多表联查、子查询、窗口函数、分组统计等核心知识点。



\## 数据表

\- \*\*students\*\*：学生表（student\_id, name, class, gender, birth\_date, enrollment\_date）

\- \*\*courses\*\*：课程表（course\_id, course\_name, credit, teacher）

\- \*\*scores\*\*：成绩表（score\_id, student\_id, course\_id, score, exam\_date）



\## 查询列表



\### 基础查询

1\. 所有学生信息

2\. 所有课程信息

3\. 所有成绩信息

4\. 每个学生平均分

5\. 每门课平均分



\### 条件查询

6\. 不及格学生

7\. 优秀学生（≥90）

8\. 三班学生

9\. 张老师课程

10\. 2024年后考试



\### 多表联查

11\. 成绩详情（学生、课程、成绩、日期

12\. 学生总成绩排名

13\. 每门课最高分

14\. 班级平均分

15\. 选修3门以上学生



\### 高级查询

16\. 窗口函数排名

17\. 课程内排名

18\. 高于平均分

19\. 成绩分布统计

20\. 最近一次考试
